# طبقة الخدمات (Services Layer)

تحتوي هذه المجلّدة على خدمات منطق الأعمال الموحّدة التي تستخدمها واجهة المستخدم
والمساعد الذكي معاً (مصدر واحد للحقيقة للعمليات).

سيتم لاحقاً إضافة:

- `AuthService` — المصادقة والصلاحيات
- `BillingService` — الفوترة والتسعير بالقياس
- `CustomerService` — إدارة العملاء
- `SupplierService` — إدارة الموردين
- `ExpenseService` — المصروفات
- `InventoryService` — المخزون وتنبيهات النقص
- `InstallationService` — مهام التركيب والفنيين
- `ReportService` — التقارير والتصدير
- `AssistantService` — المساعد الذكي

> هذا الملف عنصر نائب لتثبيت بنية المجلّد (المهمة 1.1) ويُستبدل عند تنفيذ الخدمات.
